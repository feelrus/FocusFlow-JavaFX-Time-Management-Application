# ⏱️ FocusFlow – JavaFX Time Management Application

FocusFlow is a desktop productivity timer built with JavaFX to help users manage focused work sessions using a simple countdown system. It tracks focus cycles, total study time, and provides audio alerts to support structured productivity habits.

---

## 🚀 Features

- ⏳ Customizable countdown timer (default: 5 minutes)
- 🔁 Automatic cycle tracking (focus sessions)
- 📊 Productivity statistics:
  - Completed cycles
  - Total minutes focused
  - Total hours accumulated
- 🔔 Audio alarm notification at the end of each session
- 🔇 Mute toggle for silent mode
- 🎨 Clean JavaFX UI with styled layout
- ⚡ Lightweight desktop application

---

## 🛠️ Tech Stack

- Java 17
- JavaFX (FXML, Controls, Media)
- Maven (project build automation)
- IntelliJ IDEA

---

## ▶️ How to Run

### Option 1: Using Maven Wrapper

```bash
./mvnw clean javafx:run
```

### Option 1: Using Maven Wrapper
```bash
1. Open IntelliJ IDEA
2. Click File → Open
3. Select the FocusFlow project folder
4. Wait for Maven to load dependencies
5. Ensure Java 17 is selected:
   - File → Project Structure → Project SDK → Java 17

6. Open:
   src/main/java/com/example/focusflow/HelloApplication.java

7. Click the Run (▶) button
   OR right-click → Run 'HelloApplication.main()'
```
